# 습관 리스트 (Habit List)

> 사용자들이 더 손쉽게 습관을 생성하게 도와주는 애플리케이션입니다! </br>
> 기획부터 디자인, 개발 등 혼자 진행한 프로젝트 입니다. :D

</br>

## 🛠️ 개발 정보
- 사용 언어 : kotlin
- MVVM, Room, Databinding, LiveData, RecyclerView, AlarmManager, Notification

</br>

## 📖 기능 정의
1. 습관 체크하기
2. 습관 완료 후 커멘트 작성
3. 완료 습관 기록 목록
4. 푸시 알림

</br>
